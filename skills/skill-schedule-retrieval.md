# Skill: Schedule Retrieval

How to find and select the correct PATCO schedule for the current date and time.

---

## Step 1 — Navigate to the PATCO schedules page

Open: `https://www.ridepatco.org/schedules/schedules.asp`

The mobile version (`https://ridepatco.org/m/schedules.asp`) is also acceptable if it is easier to parse. Both expose timetable effective dates, travel alerts, and special schedule notices.

---

## Step 2 — Check for an active special schedule

Before reading any standard timetable, scan the page for:

- A "special schedule" section or notice
- Travel alerts that reference altered service
- Any date-specific overrides for today

**Priority rule:** If a special schedule is active for the current date and time, use it. It overrides the standard timetable entirely.

If multiple special schedule notices appear, use the one most specific to the current date/time and service conditions.

If the page explicitly states no special schedules are in effect, proceed to Step 3.

---

## Step 3 — Identify the current standard timetable

If no special schedule applies:

- Look for the timetable marked with the latest effective date that is currently in force.
- If multiple timetable entries are listed, use the one whose effective date range covers today.
- Do not assume an older timetable is valid because it is easier to read.

---

## Step 4 — Open the applicable timetable

- If the timetable is a PDF, open it only after confirming it is the currently effective schedule.
- If the schedule wording is ambiguous about whether it applies now, note the ambiguity and explain what was used.
- If the site is unavailable, report that clearly — do not guess.

---

## Decision Priority

1. Active special schedule for the current date/time
2. Standard timetable with the latest effective date currently in force
3. If neither can be confirmed, state uncertainty and do not provide times

---

## Freshness Rules

- Re-check the live site on every request.
- Never rely on previously seen schedule content.
- Treat the PATCO site as authoritative over any memory or assumption.
